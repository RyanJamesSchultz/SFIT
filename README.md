# SFIT

I present a Matlab-based enviroment, SFIT that uses 'model differencing' to intuit a prototype to the Seismogenic Fault Injection Test (SFIT).  This repository is related to a study about the same topic [Schultz et al., 2026].  This repository contains programs, scripts, and data to recreate figures and results of the prior study.

References: 
            
            R. Schultz, A.P. Rinaldi, & S. Wiemer (2026)
            Prototyping the Seismogenic Fault Injection Test
            Geophysical Research Letters, xx, XX.
            doi: xx.
